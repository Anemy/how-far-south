[How Far South](https://howfarsouth.com)

## Installation

Install node js from https://nodejs.org/en/download/

```bash
$ # Clone this repo and navigate to the cloned folder.
$ npm install
```

## Development

```bash
$ npm start
$ browser https://localhost:3000
```
